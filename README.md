## 功能列表
+ 发送短信接口
+ 其他接口

## Installation

```sh
$ npm install apis-aliyun
```

## Use with koa

```js
import Dayu from 'apis-aliyun';
// init:
var Dayu = new Dayu ();

// get result:
var result = await lottery.getResultAfterTime (new Date(), 'cqssc');
```
